Cloud as a Service

What is SaaS?

SaaS stands for Software as a Service. It allows users to access software through the internet without installation.

Benefits of SaaS

  • Cost Savings
  • Scalability
  • Accessibility
  • Easy Updates

SaaS Maturity Model

  1. Ad Hoc
  2. Defined
  3. Managed
  4. Optimized
